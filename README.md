# blooket-hack
DISCLAMER: ONLY AVAILABLE ON ONLINE IDE LIKE VSCODE OR REPLIT

# Lets start with the base:
First step:
```
mkdir blooket-bot
cd blooket-bot
 ```
Then you gotta:
```
npm init -y
```
Here is the next step:
```
npm install blooketjs
```
# Now, set up the token:
```
npm install dotenv
```
# Finally, let's run the bot:
```
node hack.js
```
